Sample lecture PDFs for quiz generation
=======================================

Included file (after npm run sample-pdf):
  odontogenic-oral-pathology-lecture.pdf

To CHOOSE / CHANGE the lecture text:
  1. Edit lecture-content.json (title + paragraphs array)
  2. Run: npm run sample-pdf
  3. Faculty → PDF → Quiz generator → "Use sample lecture PDF"

Or upload any of your own PDFs on the faculty upload page.

The JSON paragraphs should be clear factual sentences (50+ characters each)
for best automatic quiz generation.
