Demo tooth GLB gallery
======================

These files are for live demos only. The app default stays in public/models/single_tooth.glb.

During a presentation you can either:
  • Use Student → 3D models → Demo gallery buttons, or
  • Choose file and browse to this folder on disk:
      oral-pathology-edu\public\models\demo-gallery

Add or replace .glb files here, then update src/data/toothDemoGallery.ts if you want new quick-load buttons.
