AR lab sample images
=====================

Purpose-built front/occlusal jaw references that align with the AR overlay
(ar-jaw-sketch.svg). All bundled files are SVG illustrations (no stock photos).

Bundled samples:
  jaw-front-anterior.svg   — anterior view (recommended default)
  jaw-occlusal-arch.svg    — occlusal arch
  impacted-molar-clinical.svg — impacted crown teaching view
  edentulous-ridge-model.svg  — alveolar ridge / center marker

To add more: place .svg or .png in this folder and add an entry to index.json.
